# Git Add Commit Command

Lakukan operasi git add dan git commit otomatis dengan konvensi pesan commit yang spesifik.

**Tugas:**
1. Check git status untuk melihat file yang berubah
2. Analyze setiap file untuk menentukan jenis perubahan dan scope
3. Group file yang terkait langsung atau buat commit individual per file
4. Execute git add per file/group
5. Execute git commit dengan pesan yang sesuai konvensi
