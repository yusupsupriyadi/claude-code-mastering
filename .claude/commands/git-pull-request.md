# Create Pull Request Command

Membuat pull request otomatis untuk branch yang sedang aktif menggunakan GitHub CLI dengan konvensi yang profesional.

**Tugas:**
1. Check current branch status dan git status
2. Analyze commit history dari base branch hingga HEAD
3. Generate title yang relevan dengan branch name dan changes
4. Create comprehensive description berdasarkan changes
5. Execute gh pr create dengan title dan description yang generated
6. Verify PR creation dan provide URL

**Workflow:**
1. Jalankan `git status` dan `git log` untuk analyze changes
2. Determine base branch (biasanya main/master)
3. Generate appropriate title berdasarkan konvensi
4. Create detailed description dengan format yang ditentukan
5. Execute `gh pr create --title "title" --body "description"`
6. Confirm successful creation dan tampilkan PR URL
